# Profitability Thresholds in Indian Quick Commerce: A Dark Store Unit Economics Analysis

## Research question
What is the exact mathematical tipping point required for a single dark store 
to achieve sustainable EBITDA profitability, and how does that differ between 
a representative Tier-1 and Tier-2 Indian city?

## Overview
A quantitative analysis of quick-commerce dark store economics in India, 
examining platforms like Blinkit, Zepto, and Swiggy Instamart. The project 
models the precise intersection of order volume, SKU mix, and logistics costs 
that separates a loss-making dark store from a profitable one.

## Models
- Unit economics P&L waterfall
- Poisson demand modeling
- Sensitivity and correlation analysis

## Tools
Python — pandas, numpy, scipy, matplotlib, seaborn
